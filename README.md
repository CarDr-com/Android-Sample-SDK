# Android-Sample-SDK
Sample app to demonstrate Android SDK integration


SDK Version: 1.0.22

This repository contains a sample Android application that demonstrates integration with the OBDIQ Android SDK (version 1.0.22) provided by CarDr.

The sample is intended as a reference implementation for developers who want to build Android applications that communicate with vehicles using the OBD-II interface. It highlights the recommended usage patterns, lifecycle handling, and overall integration flow for the OBDIQ SDK.
